# 📝 Python To-Do List Application

A simple command-line To-Do List application built using Python.

## Features

-   Add new tasks
-   View all tasks
-   Delete tasks
-   Exit the application
-   Handles invalid input

## Technologies Used

-   Python 3

## Project Structure

    todo-list/
    │── todo.py
    └── README.md

## How to Run

``` bash
python todo.py
```
